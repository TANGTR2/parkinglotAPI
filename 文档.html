2、创建停车场

method：Post
path:"/parkingLots"
request:{ParkingLot parkingLot}
        {"plId": 3,"plName": "西方停车场","plSize": 1}
response:
{
    "plId": 3,
    "plName": "西方停车场",
    "plSize": 1
}

4、创建停车小弟

method：Post
path:"/parkingBoys"
request:{ParkingBoy parkingBoy}
        {"pbId": 5,"parkingLots": []}
response:
{
    "pbId": 5,
    "parkingLots": []
}

5、给停车小弟安排停车场

method：Post
path:"/parkingBoys/{pbId}/parkingLots"
      http://localhost:8080/parkingBoys/1/parkingLots
request:ParkingLot parkingLot, int pbId
       {"plId": 3,"plName": "西方停车场","plSize": 1}
response:
{
    "pbId": 1,
    "parkingLots": [
        {
            "plId": 3,
            "plName": "西方停车场",
            "plSize": 1
        }
    ]
}
